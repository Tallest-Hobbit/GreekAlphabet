# Image Sound Swiper

A Flutter app that lets the user:

- swipe left and right through a set of images
- tap the upper half of an image to play one sound
- tap the lower half of an image to play a different sound

Each image has its own unique top and bottom sound.

## Setup

1. Install Flutter
2. Create or copy your image/audio assets into:

```text
assets/images/
assets/sounds/
